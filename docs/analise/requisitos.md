# 📌 Requisitos do Sistema

## Requisitos Funcionais

- RF1: Ler vídeo `video.avi`
- RF2: Detetar laranjas em cada frame
- RF3: Contar laranjas na frame atual
- RF4: Manter contagem total acumulada
- RF5: Calcular área de cada laranja
- RF6: Calcular perímetro
- RF7: Determinar bounding box
- RF8: Calcular centro de gravidade
- RF9: Determinar calibre
- RF10: Determinar categoria
- RF11: Mostrar informação sobre o vídeo

## Requisitos Técnicos

- RT1: Implementação em C/C++
- RT2: Processamento frame a frame
- RT3: Execução em tempo real
- RT4: Suporte para 1280x720
- RT5: Conversão px → mm
- RT6: Sistema modular e compreensível

## Restrições

- RS1: Uso limitado de OpenCV
- RS2: Sem bibliotecas externas
- RS3: Sem código externo (GitHub)
- RS4: Defesa oral obrigatória
- RS5: Todos os elementos devem dominar o código
